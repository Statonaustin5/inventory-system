# Smart Inventory Management System

A Python desktop app for tracking product inventory, built with Tkinter, SQLite, Pandas, and NumPy.

## Features
- Add, edit, and delete products through a GUI
- SQLite database for persistent storage
- Automatic restock alerts based on configurable thresholds
- Demand forecasting using OLS linear regression (ŷ = β₀ + β₁x)
- Weekly sales reports with week-over-week trends

## Setup
```
pip install numpy pandas
python main.py
```
